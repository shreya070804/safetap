# 🚨 SafeTap – Smart Safety & Emergency Response System

SafeTap is an AI-powered safety platform designed to provide real-time emergency assistance through SOS alerts, location tracking, and health monitoring.

---

## 🔥 Features

- 🚨 One-click SOS alert system
- 📍 Live location sharing via Google Maps
- 👨‍👩‍👧 Emergency contact (Guardian) system
- ❤️ Health monitoring dashboard (Heart rate, steps)
- 📧 Email alert system using EmailJS
- 🔒 Secure medical record storage
- 🧠 AI-enhanced safety insights (future scope)

---

## 🛠️ Tech Stack

- Frontend: React + Vite
- Backend: Supabase
- Maps: Leaflet / OpenStreetMap
- Email Service: EmailJS
